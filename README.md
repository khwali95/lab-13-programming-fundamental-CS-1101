# lab-13-programming-fundamental-CS-1101
task 1: demonstrates fundamental pointer operations storing addresses and accessing values via the dereference operator(*)
task 2: show how pointers can indirectly modify variables, enabling "pass-by-refrence"behaviour.
task 3: demonstrates that array names decay to pointers and how pointer arithmetic (ptr+1) allows sequential array traversal.
task 4: implements a classic "swap" function using pointers to modify orginal variables.
task 5: Emphasizes pointer initialiazation and safety checks to prevent dereferencing invald memory.
task 6: demonstrates generic pointers that can point to any data type but require explicit casting before dareferencing.
